# REQUISIT EXTRA PER PDF2IMAGE (Linux, Ubuntu, Codespaces)

Per convertir PDFs a imatges amb pdf2image (OCR), cal tenir Poppler instal·lat al sistema.

## Instal·lació a Linux (Ubuntu, Codespaces, WSL...)

Executa al terminal:

```
sudo apt-get update
sudo apt-get install -y poppler-utils
```

Després ja podràs pujar PDFs i fer OCR correctament.

---

Si treballes a Mac, pots instal·lar Poppler amb Homebrew:
```
brew install poppler
```

Si tens dubtes, consulta la documentació de pdf2image o demana ajuda!
