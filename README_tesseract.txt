# REQUISIT EXTRA PER pytesseract (Linux, Ubuntu, Codespaces)

Per fer OCR amb pytesseract, cal tenir el binari Tesseract instal·lat al sistema.

## Instal·lació a Linux (Ubuntu, Codespaces, WSL...)

Executa al terminal:

```
sudo apt-get update
sudo apt-get install -y tesseract-ocr
```

Després ja podràs fer OCR sobre imatges i PDFs sense errors.

---

Si treballes a Mac, pots instal·lar Tesseract amb Homebrew:
```
brew install tesseract
```

Si tens dubtes, consulta la documentació de pytesseract o demana ajuda!
